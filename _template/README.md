# SCC Device Portal Guide Template

This folder is the master template for creating new SCC Device Portal guides.

## How to create a new guide

1. Copy the entire `_template` folder.
2. Rename the copied folder, for example `android`, `returns`, `collection`, or `smart-lockers`.
3. Keep the filenames as:
   - `index.html`
   - `style.css`
4. Add screenshots into the `images` folder.
5. Update the guide title, intro text, overview cards, section banners, step text, image filenames and alt text.
6. Test the guide using the folder URL with a trailing slash, for example:
   - `/android/`
   - `/returns/`

## Notes

- Keep the class names the same to preserve styling.
- Use `<strong>` for button names, app names and important user choices.
- Delete any step cards or info cards that are not needed.
- Keep the Need Help and Finished cards at the bottom of every guide.
