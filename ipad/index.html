SALFORD CITY COUNCIL IT SERVICES

iPad Setup Guide

Follow this guide to set up your SCC managed iPad.

Estimated time: 10–15 minutes Difficulty: Easy For SCC iPads

What you will set up

1 Initial iPad setup
2 SCC device enrolment
3 Company Portal access
4 Microsoft Defender

Before you start

Make sure the iPad has enough battery and keep it connected to Wi-Fi during setup.

If you have requested an iPad with Wi-Fi and Cellular, mobile data may become active after the device is enrolled. Some newer iPads no longer use a physical SIM card.

PART 1

Connect Your iPad

Start the iPad and connect it to Wi-Fi.
Step 1

Start Setup

Swipe up from the Hello screen to begin.

Step 2

Select Country or Region

Select United Kingdom.

Step 3

Appearance

Leave the appearance setting as Default, then select Continue.

Step 4

Set Up Without Another Device

Select Set Up Without Another Device.

Step 5

Connect to Wi-Fi

Select SCC-Setup Wi-Fi.

If setting up away from an SCC office, connect to your home Wi-Fi.

PART 2

Enrol Your Device

Register the iPad as an SCC managed device.
Step 6

Device Management

The iPad should show that it is owned by Salford City Council.

Select Enrol this iPad.

Step 7

Sign In

Sign in using your Salford City Council email address.

Follow any authentication prompts if they appear.

Step 8

Touch ID

You can set up Touch ID now or select Set Up Touch ID Later.

Step 9

Create an iPad Passcode

Create a passcode to unlock the iPad.

Do not share your passcode with anyone.

Step 10

Turn On Location Services

Select Turn On Location Services.

This helps with location-based services and may assist IT if a device is lost.

Step 11

Multitasking and Windowing

Select your preferred option, then select Continue.

PART 3

Complete Company Setup

Open Company Portal and finish SCC access setup.
Step 12

Home Screen

Once the iPad reaches the Home Screen, setup has completed successfully.

Open Company Portal.

Step 13

Company Portal

Select Sign in.

Step 14

Set Up SCC Access

Select Begin.

Step 15

Select Device Category

Select SCC Default iPad, then select Continue.

Step 16

Company Setup Complete

When you see You're all set!, select Done.

PART 4

Open Microsoft Defender

Return to the Home Screen and complete Microsoft Defender setup.
Step 17

Open Microsoft Defender

Return to the Home Screen and open the MS Defender app. If prompted, select your Salford City Council account to continue.

Step 18

Complete Microsoft Defender Setup

Read and accept the Microsoft Defender Terms of Use, then select Accept.

When prompted:

  • Select Allow to add the VPN configuration.
  • Select Allow to enable notifications.

Your SCC iPad is now fully configured and ready to use.

Setup Complete

Your SCC iPad is now set up and ready to use.
